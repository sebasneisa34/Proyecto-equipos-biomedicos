# Proyecto Integrador

**Integrantes**  1 - 5 personas

**Asignaturas**  Programación Orientada a Objetos, Bases de Datos

**Tema Integrador**  Sistema para control de ingreso de equipos tecnológicos y biomédicos externos.

## Contexto y Propósito 

Desarrollar un sistema que permita registrar, monitorear y 
controlar el ingreso de equipos tecnológicos y biomédicos
externos al hospital, con el fin de garantizar la trazabilidad
seguridad operativa y cumplimiento de los protocolos institucionales 
establecidos

## Aplicación de Clean Architecture

Los siguientes paquetes cumplen con el concepto
de arquitectura limpia:

- edu.usta.domain
 - entities
 - enums
 - repositories
- edu.usta.infraestructure
